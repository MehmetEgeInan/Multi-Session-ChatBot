## 🌟 Öne Çıkan Özellikler / Key Features
- 🗂️ **Oturum Yönetimi** - Her kullanıcı için bağımsız sohbet geçmişi  
  **Session Management** - Independent chat history per user
- 💾 **SQLite Veritabanı** - Tüm mesajlar kalıcı olarak saklanır  
  **SQLite Database** - Persistent message storage
- 🤖 **OpenAI Entegrasyonu** - GPT-3.5-turbo ile akıllı yanıtlar  
  **OpenAI Integration** - Smart responses with GPT-3.5-turbo
- 📱 **Responsive Arayüz** - Streamlit ile modern kullanıcı deneyimi  
  **Responsive UI** - Modern interface with Streamlit
